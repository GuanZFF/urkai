# 归并排序程序

这是一个使用Python实现的归并排序算法，专门用于对整数列表进行排序。

## 文件说明

- `merge_sort.py` - 归并排序的核心实现
- `sort_cli.py` - 命令行交互工具

## 功能特点

1. **完整的归并排序实现**：
   - 递归分割数组
   - 合并已排序的子数组
   - 时间复杂度：O(n log n)

2. **多种输入方式**：
   - 命令行参数
   - 交互式输入
   - 支持多种分隔符（空格、逗号、分号）

3. **错误处理**：
   - 自动跳过非数字输入
   - 空数组处理
   - 单个元素处理

## 使用方法

### 1. 直接运行演示程序

```bash
python merge_sort.py
```

这会显示多个测试用例的排序结果。

### 2. 使用命令行工具

**交互模式：**
```bash
python sort_cli.py
```

**直接排序：**
```bash
python sort_cli.py 5 2 8 1 9
# 或使用其他分隔符
python sort_cli.py 5,2,8,1,9
python sort_cli.py "5;2;8;1;9"
```

### 3. 作为模块导入

```python
from merge_sort import merge_sort

# 使用归并排序
numbers = [64, 34, 25, 12, 22, 11, 90]
sorted_numbers = merge_sort(numbers)
print(sorted_numbers)  # [11, 12, 22, 25, 34, 64, 90]
```

## 算法原理

归并排序采用分治策略：
1. **分割**：将数组递归地分成两半，直到每个子数组只有一个元素
2. **排序**：单个元素的数组自然是有序的
3. **合并**：将两个已排序的子数组合并成一个有序数组

## 性能特点

- **时间复杂度**：O(n log n)（最好、最坏、平均情况）
- **空间复杂度**：O(n)（需要额外的存储空间）
- **稳定性**：稳定排序算法（相等元素的相对位置不变）
